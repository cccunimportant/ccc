流程如下

  • 撰寫一堆 markdown 稿件
  • 寫一個 makefile
  • 呼叫 pandoc 將 markdown 轉為 html 與 epub
  • 再叫 calibre 將 epub 轉為 pdf

後來發現

  • 網頁內的 latex 數學式

所以我就

用來將

  • 然後就可以放在 pdf 裏了
